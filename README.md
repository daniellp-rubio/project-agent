🚀 Overview
This project is a scalable conversational AI system designed to help clinics turn incoming chats into qualified appointments.

Instead of acting as a simple chatbot, the system analyzes each conversation to identify user intent, objections, and conversion opportunities—helping businesses not only automate responses but improve their sales process.

🎯 Key Features
AI-powered conversations
Natural, goal-oriented interactions using LLMs via OpenRouter
Lead qualification
Automatically filters and identifies high-intent patients
Appointment-driven flow
Designed to convert conversations into scheduled visits
Conversation analytics
Detects:
Intent (interested / curious / not qualified)
Common objections (price, fear, trust)
Frequent questions
Actionable insights dashboard
Helps clinics understand why leads don’t convert and how to fix it

🧠 Core Value
This is not just a chatbot.
It’s a system that turns conversations into revenue and insights.

🛠️ Tech Stack
Next.js
Supabase
OpenRouter (LLMs)

📌 Use Case
Built initially for aesthetic clinics, focused on:

Automating patient interactions
Reducing manual WhatsApp workload
Increasing appointment conversion rates
